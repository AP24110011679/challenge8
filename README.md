# Smart City Data Intelligence System
Overview
This project simulates a smart city system that monitors **traffic, air quality, and energy usage** across zones and analyzes overall city risk.
Features
* Generates random city data
* Classifies zones (High Risk, Energy Critical, Safe, Moderate)
* Calculates risk score using a custom formula
* Sorts data using Bubble Sort
* Detects patterns and clusters
* Gives final city status
 Tech Used
* Python
* Pandas
* NumPy
 How to Run
```bash
pip install pandas numpy
python smart_city.py
```
 Output
* Data table
* Top risk zones
* Risk statistics
* Final decision (City Stable / Alert / Emergency)


This project is for academic and educational purposes.
